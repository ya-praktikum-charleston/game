## Getting Started

### .env
```
COMPOSE_PROJECT_NAME=
EXPRESS_PORT=
NODE_ENV=
APP_URL=
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_PORT=
DB_HOST=
```

### Запуск (development)

```
npm run start
```

### Тесты

```
npm run test
```

### Сборка (production)

```
npm run build
```
### Старт сервера (production)

```
npm run start:prod
```
